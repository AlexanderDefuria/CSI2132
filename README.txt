INSTALLATION INSTRUCTIONS

Debian Based System
sudo apt-get install python3
sudo apt-get install python3-venv
python3 -m venv ./venv
source ./venv/bin/activate
pip3 install -r ./requirements.txt
python3 app.py

------ or ------

pip3 install -r ./requirements.txt
python3 app.py

-----------------

Windows is a similar process just ensure that all dependencies are met and the
system is running on python 3.7 and run app.py.

Navigate to http://127.0.0.1:5000
Use application per presentation instructions.
